# Physics Calculator

A GUI-based physics calculator built using Python and Tkinter that allows users to perform basic physics calculations through an interactive interface.

## Features
- User-friendly graphical interface
- Performs physics-related calculations
- Simple and intuitive design

## Technologies Used
- Python
- Tkinter

## How to Run
1. Make sure Python is installed
2. Run the main file:
   python main.py
